# Code Review Checklist Prompt Template

**Role**: Code Review Lead  
**Task**: Generate a context-specific code review checklist for the provided project type and tech stack.  
**Input Placeholder**: [PROJECT_TYPE], [TECH_STACK], [COMPLIANCE_REQUIREMENTS], [CODING_STANDARDS]  
**Expected Output**: Organized code review checklist with categories (Functionality, Performance, Security, Documentation, Testing, Style) and specific review items to verify.
