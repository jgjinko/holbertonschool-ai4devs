# StockSync API - Development Prototype

This is the core inventory API for the StockSync platform, built with Express.js. It supports multi-channel inventory tracking, pagination, and full CRUD operations.

## Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- NPM (included with Node)
