# Node.js Express Template (AI-Enhanced)

## Features
- **Security-First**: Copilot settings enforce input validation (Zod/Joi) for all endpoints.
- **Auto-Docs**: Specialized workflow for generating Swagger/OpenAPI specifications.
- **Middleware Logic**: Pre-configured rules for JWT and Error Handling middleware.

## Setup
1. Copy contents to project root.
2. Run `npm install`.
3. Run Task: `AI Documentation Pass` to update JSDoc headers.
